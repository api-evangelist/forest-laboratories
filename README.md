# Forest Laboratories

Forest Laboratories was a pharmaceutical company that developed, manufactured, and sold branded prescription drugs. It was acquired by Actavis in 2014, which later rebranded as Allergan and is now part of AbbVie. As a defunct entity, Forest Laboratories does not publish any APIs or developer resources.
